Construir una imagen con node

  • Crear un Dockerfile

  • Docker File

    
            FROM node:12
            WORKDIR /app
            COPY package*.json ./
            RUN npm install
            COPY . .
            CMD ["npm", "run", "dev"]

  • Crear un dockerignore

    
            node_modules
            npm-debug.log

  • Creacion de la imagen

    
            docker build -t nombre-app

  • Ejecutar la imagen

    
            //interactivo
            docker run -it -p 4000:3000 nombre-app
            //servicio
            docker run -d -p 4000:3000 nombre-app
