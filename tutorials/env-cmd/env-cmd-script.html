Como usar env-cmd en un script de npm

Si quieres usarlo directamente, y que las variables de ambiente se compartan en la misma estancia de ejecion


        file: .env
        TEST=EPA PANA
        npm script
        "test": "env-cmd --use-shell \"echo %TEST% && echo olis && echo %TEST%\""

Resultado


        EPA PANA 
        olis 
        EPA PANA
