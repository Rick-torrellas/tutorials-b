Actualizacion automatica para ubuntu

Instalacion

apt install unattended-upgrades

Verify the installation:

systemctl status unattended-upgrades

Configuracion

sudo dpkg-reconfigure --priority=low unattended-upgrades

Testing Automatic Upgrades

The dry run command ensures that the update is only a simulation and that no actual changes will take place. Use the command:

sudo unattended-upgrades --dry-run --debug

log file:

/var/log/unattended-upgrades/unattended-upgrades.log

(Optional Info)

Accessing the Configuration File

sudo nano /etc/apt/apt.conf.d/50unattended-upgrades

Blacklist Packages

Accesing Automatic Upgrades File

sudo nano /etc/apt/apt.conf.d/20auto-upgrades
