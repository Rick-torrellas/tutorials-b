Ejemplo de acciones

Hacer un escaneo desde la consola


        docker run -it sundowndev/phoneinfoga scan -n cotrycode/number
        docker run -it sundowndev/phoneinfoga scan -n +584241568747

Ejecutarlo desde una GUI - interfas de usuario


        docker run -it -p 8080:8080 sundowndev/phoneinfoga serve -p 8080

Esto hara correr el comando en el puerto 80, usando una interfas de usuario
