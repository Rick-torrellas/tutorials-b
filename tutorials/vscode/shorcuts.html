Seleccion

Esconder/Mostrar el nav bar


        control + b

Esconder/Mostrar la consola

control + `

settings

control + ,

Abrir un archivo rapidamente

control + p

Seleccionar elementos selecionados uno a uno

control + d

Seleccionar todos los elementos seleccionados

control + f2 o control + shift + l

tab atraves de los archivos abiertos

control + tab

selecionar los archivos abiertos segun su posicion

alt + posision numerica(1,2,3,etc)

mover el archivo a una ventana dividida

control + alt + -> (flecha derecha)

duplicar el archivo en una ventana dividida

control + \

cerrar archivos

control + w

buscar palabra dentro del archivo

control + f

bsucar y remplazar

control + h

bsucar y remplazar global

saltar a una linea

control + g

abrir los paleta de comandos de vscode

comentar una linea

control + /

control + shift + a
