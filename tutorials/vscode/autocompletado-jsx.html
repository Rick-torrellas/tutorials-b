Autocompletar archivos js usando jsx

La forma más directa de obtener el autocompletado JSX/HTML en sus proyectos React es agregar esto a la configuración del usuario o la configuración del espacio de trabajo (<project-path>/.vscode/settings.json):


        "emmet.includeLanguages": {
            "javascript": "javascriptreact"
          },
          "emmet.triggerExpansionOnTab": true
