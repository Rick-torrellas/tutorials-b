mongod parametros

--logpath

Para indicar la ruta donde se va a colocar el log

--logpath F:\pupu\sesions\log\mongo.log

--dbpath

Para indicar donde se va a guardar la base de datos

F:\pupu\sesions\data\db

--config

Para indicar el archivo de configuracion

--config ruta\mongod.conf

--auth

Para iniciar mongo y que sea necesario la autentificacion
