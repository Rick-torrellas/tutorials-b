Exportar una base de datos desde Mongodb

mongodump

Exportar una base de datos, que se encuentra dentro de Mongodb, a un lugar.

Forma basica

mongodump

De esta manera realizara un respaldo de todas las bases de datos, por defecto lo hara en el directorio actual, en una carpeta llamada dump/

Parametros mas importantes

-d,--db

El nombre de la base de datos que quieres exportar, si no se espesifica copia todas las bases de datos.

mongodump -d=vagina

--collection,-c

Espesica la coleccion que quieres exportar, si no espesifica se exportaran todas las colecciones, me imagino que para usar esta opcion se debe espesificar una base de datos.

mongodump -d=vagina -c=vaginas-peludas

--gzip

Si quieres que las colecciones exportadas esten comprimidas en rar

mongodump --gzip

--out

La ruta donde quieres que se coloque la exportacion, si esta opcion no es dada, se exportara en una carpeta llamada dump, en el directorio actual.

mongodump --out=database/vagina/

--host

La direccion ip y el puerto, donde vamos a hacer el respaldo.

mongodump --host=ip:puerto
mongodump --host=190.79.76.206:27017 (el puerto por defecto)

--username,-u

El usuario de mongo, en caso de tener uno
