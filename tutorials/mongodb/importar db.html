Importar base de datos a Mongodb

mongorestore

Importa una/s base de datos que tengas al sistema de Mongodb.

Manera basica

mongorestore dump/

Esto importa todas las bases de datos que tengas en la carpeta dump/, el nombre que se le pondra es el nombre de la carpeta contedora de la db. creo :D

--host

La ip y el puerto donde se encuentra el sitema mongodb, donde vas a colocar la db

mongorestore --host=ip:puerto dump/

--username,-u

EL usuario del sistema de mongodb.

--password,-p

La contrase√±a del usuario del sistema de mongodb.

--db,-d

El nombre de la base de datos que vas a realizar la importacion, no se usarlo el todo, osea si funciona,pero me muestra un mensaje diciendome que pronto va a hacer decapricado y que use --nsInclude.

NOTA: En este caso vamos a seleccionar la base de datos que vamos a importar

mongorestore -d=vaginas dump/nalga

--collection,-c

Espesifica la coleccion que va a ser importada

mongorestore --db=reporting --collection=employeesalaries dump/test/salaries.bson

--gzip

Importa base de datos comprimidas en rar.

--authenticationDatabase

Espesifica la base de datos donde fue creado el usuario para la autentificacion.

--authenticationDatabase "basedatos"
