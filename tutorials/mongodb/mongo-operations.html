mongo Operaciones Basicas

Operaciones basicas

Crear db

Solo hace falta posicioarte en ella, se creara automanticamente sin estar creada previamente.

use base-datos

Crear coleccion o tabla

Son como las tablas de SQL

db.nombre-tabla.save({id: 1, nombre: "el coño de tu madre", guebonada: "algo"})

Mostrar colecciones

show collections

Mostrar usuarios

show users

Mostrar registros de una coleccion

Con este te muestra todas las colecciones que coincidan con ese criterio.

db.coleccion.find()

Para encontrar uno o varios valores

db.tabla.find({nombre: "nalga","guebonada":"pija"})

Eliminar una coleccion

db.nombre-coleccion.drop()

Remover documentos de una coleccion

Remueve todos los documentos dentro de una query

db.products.remove({ _id: 1, category: "café", status: "A" })
