Como clonar directorios y archivos

lo primero es habilitar esta funcionalidad en el repositorio vacio que hemos creado con git init

git config core.sparsecheckout true

luego es crear un archivo en .git.info llamado sparse-checkout, el cual tendra todas las carpetas y/o archivos que se quieran descargar, ejemplo:

echo /css/ >> .git/info/sparse-checkout

NOTA: es importante recordar que esto se puede hacer una sola vez, antes de hacer el pull, ya que si hacemos el pull y luego volvemos a agregar carpetas/archivos al sparse-checkout, ya no traera los nuevos archivos.

Por ultimo hacemos un pull al repositorio:

git pull --depth=1 origin master
