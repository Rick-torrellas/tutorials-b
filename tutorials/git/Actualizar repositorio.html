Actualizar tu repositorio

En caso de que tengas un repositorio que necesita ser actualizado, ya sea que actualizaste el codigo desde github o localmente, y quieres traerte esos cambios al repositorio actual, puedes hacerlo de esta manera.

En caso de que sea un repositorio clonado cn git clone

git pull

O puede que sea el repositorio local y hiziste los camibios desde github o desde otro repositorio local

git pull github/origin (o como sea que lo llamaste)
