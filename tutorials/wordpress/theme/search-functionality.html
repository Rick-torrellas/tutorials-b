            file: front-page.php
            <?php get_search_form() ?>

Mostrar los resultados


        file: search.php
        <?php if (have_posts()): while(have_posts()): the_post(); ?>
<?php get_template_part('includes/section','thumbnailS'); ?>

<?php the_title(); ?>

<?php the_excerpt(); ?>
<a href="<?php the_permalink(); ?>">Read more

<?php endwhile; else: ?> 
    
The are no results for: ''

    <?php endif;?>

Basicamente el archivo puede ser muy parecido al front-page por ejemplo. solo que si quieres puedes usar get_search_query(), para mostrar el query usado por ejemplo.

Custom search form


        file: searchform.php
    <form action="<?php home_url();?>" method="get">
    <label for="search">Search</label>
    <input type="text" name="s" id="search" value="<?php the_search_query(); ?>" required>
    <button type="submit">Search!</button>
    </form>

el action debe estar dirigido a la pagina raiz del proyecto de wordpress como tal.
