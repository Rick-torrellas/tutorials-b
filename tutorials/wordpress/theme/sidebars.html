Sidebars

Pre-requisitos

Para que funcionen los sidebars hay que habilitar los widgets.

add_theme_support('widgets');

Declaracion


        file: functions.php
        function xample_add_sidebars( ) {
            register_sidebar(
                [
                    'name' => 'Page Sidebar',
                    'id' => 'page-sidebar',
                    'before_title' => '
',
                    'after_title' => '
'
                ]
                );
                register_sidebar(
                    [
                        'name' => 'Blog Sidebar',
                        'id' => 'blog-sidebar',
                        'before_title' => '
',
                        'after_title' => '
'
                    ]
                    );
    }
    add_action('widgets_init','xample_add_sidebars');

implementacion


        <?php if(is_active_sidebar('page-sidebar')): ?>
    <?php dynamic_sidebar('page-sidebar') ?>
    <?php endif; ?>

Configuracion en wordpress

Para configurar cada sidebar declarado en functions.php, hay que hacerlo manualmente en wordpress.

Apparience/widgets/

Ahi te apareceran todos ;ps widgets declarados anteriormente. y podras configurarlo.
