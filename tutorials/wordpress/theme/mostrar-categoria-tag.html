archive.php / mostrar categoria - tag

La funcion de este archivo es mostrar cada categoria o tag. una a la vez, osea no muestra todas las categorias/tags.

Ejemplo basico:

En este ejemplo se muestra una categoria/tag y se muestran todos los post relacionados a el.


        file: archive.php
        <h1><?php echo single_cat_title(); ?></h1>
        <?php if (have_posts()): while(have_posts()): the_post(); ?>
        <h3><?php the_title(); ?></h3>
        <?php the_excerpt(); ?>
        <a href="<?php the_permalink(); ?>">Read more</a><br>
        <?php endwhile; else: endif;?>
