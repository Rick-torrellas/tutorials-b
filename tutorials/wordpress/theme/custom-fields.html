Activarlo

Con custom post/page

Cuando estas creando tu custom post, tiene que espesificarlo para usarlo.


        'supports' => ['title','editor','thumbnail','custom-fields']

Activarlo desde wordpress

Para habilitar los custom fields para todo tipo de post/page.

Vamos al post/page en cuestion y le damos a Screen Options/custom fields.
