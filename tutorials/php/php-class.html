private

Son propiedades/metodos que solo estaran disponibles dentro de la clase.

protected

Se usa para usar una propiedad de otra clase cuando se hereda otra clase. pero comparte el mismo principio de private, osea se puede acceder a ella dento de las clases y no fuera. Para acceder a ella se necesita crear una propiedad publica que referencie a esa propiedad.

public

Son propiedades/metodos que estaran disponibles fuera de la clase

static

Son propiedad/metodos que no necesitan ser instanciados en un objeto para poder ser usados.

$a = $this->var;

Clases


        class Person {
            public $name; 
            public static value = "Valor global";
        public function __construct($name = 'jimmy') {
            $this->name = $name
        }
        public function myFuncion() {
            return $this->name;
        }
        public function myFuncion2($stuff) {
            return $stuff;
        }
        public function myFuncion3() {
            return self::$value;
        }
        public static function setValue($newValue) {
            return self::$value == $newValue;
        }
        }
        $object = new Person('Pedro');
        $object->name;
        $object->myFuncion2('algo');
        echo Person::$value;
        echo Person::setValue('nuevo valor');

final

Cuando una clase tiene la palabra final, significa que no puede ser extendida en otra clase


        final class Nalga {
            
        }

Static espesificaciones

static properties espesificaciones

Las propiedades estaticas no pueden guardar otras variables estaticas,propiedades normales, funciones normales y funciones estaticas.

Si quieres concatenar propiedades estaticas, puedes usar una funcion estatica.

funciones estaticas espesificaciones

Las funciones estaticas no podran guardar ni propiedes o funciones normales, pero si pueden guardar tanto funciones como propiedades estaticas.

Te dice que no se puede usar $this, dentro del ambito de la funcion.

propiedades espesificaciones

Para que una propiedad pueda guardar el valor de una fucion declarada en la misma clase, necesita hacerse dentro del contructor o de otra funcion.
