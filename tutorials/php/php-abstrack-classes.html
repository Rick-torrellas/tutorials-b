Abstrack Classes

Las clases abstractas sirven para complementetar otras clases, para extender otras clases, estas no pueden ser instanciadas como un objeto.

Metodos Abstractos

Cuando defines un metodo abstracto, estas diciendo que en la clase que va a ser refrencia, necesita tener un metodo con ese nombre.


        abstract class Visa {
            public function visaPayment() {
                return "pagando";
            }
        abstract public function getPayment();
        }
        class Person extend Visa {
            public function getPayment() {
                return $this->visaPayment();
            }
        }
