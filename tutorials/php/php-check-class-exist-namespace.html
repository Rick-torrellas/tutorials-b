class_exists()

Si se quiere verificar que una clase existe en determinado namespace con la funcion class_exists, hay que tener en cuenta una sola cosa, ejemplo:


        use home\Globals;
        if (class_exists('home\\Globals')) {
            Globals::register();
        }

Si por ejemplo se intenta verificar la clase directamente asi dara error


        use home\Globals;
        if (class_exists('Globals')) {
            Globals::register();
        }
        o
        if (class_exists('home\Globals')) {
            Globals::register();
        }

Ambos dara error.
