Mostrar una variable en una string sin concatenar

En php se pueden insertar variables dentro de una string usando comillas dobles ""

NOTA: esto nada mas funciona con las funciones.


        class pi {
            public static $value=3.14159;
            public $valor;
            public $valor2;
            function __construct() {
            $this->valor = 'como estas';
            $this->valor2 = "Hola $this->valor";
            }
            public function staticValue() {
              return "$this->valor2";
            }
          }
          
          $pi = new pi();
          echo $pi->valor;
