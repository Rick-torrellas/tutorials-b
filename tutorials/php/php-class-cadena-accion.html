Encadenar acciones con una clase

Con las clases podemos hacer cadenas de acciones, al regresar la clase en si en una funcion, como en este ejemplo


        class SettingsApi
{
    public $admin_pages = [];
    
    public function addPages($pages) {
        $this->admin_pages = $pages;
      	echo $this->admin_pages;
        return $this;
    }
  	public function hola() {
    echo ' como estas';
    }
}
$x= new SettingsApi();
$x->addPages('Epa')->hola();
