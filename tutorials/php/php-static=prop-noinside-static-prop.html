Meter una propiedad estatica dentro de otra propiedad estatica


        class pi {
            public static $value=3.14159;
            public static $nalga=self::$value . 'nalga';
            public function staticValue() {
              return self::$value;
            }
          }
        echo pi::$nalga;

Esto soltara un error, una manera de acceder a esta variable es dentro de una funcion.


        class pi {
            public static $value=3.14159;
            public static function staticValue() {
              return self::$value;
            }
          }
          
          // Get static property
          echo pi::staticValue();

NOTA: no tiene por que ser estatica necesariamente.
