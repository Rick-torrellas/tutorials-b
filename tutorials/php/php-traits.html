traits

Los traits agregan metodos/funcionalidades a las clases, sin complicarse tanto con la herencia.

Los traits no pueden ser instanciados como un objeto directamente.

Los traits tambien pueden usar metodos abstractos.

Un trait y una clase no se pueden llamar igual.


        trait Pajuo {
            public $nombre;
            public function mostrarNombre(){
                echo $this->nombre;
            }
            abstract function ola($nombre);
        }
        trait Cagon {
            protected function mensaje() {
                echo "Me gusta cagar y soy:";
            }
        }
        class Persona {
            use Pajuo,Cagon;
            public function ola($nombre) {
                $this->nombre = self::mensaje() $nombre;
            }
        }
        $obj = new Persona();
        $obj->ola('Ramon');
        $obj->mostrarNombre();

Definir propiedades

Para definir proppiedades en un trait se tiene que espesificar en el ambito public, tanto como si no lo espesificas o colocas private o protected, causara un fata error.


        trait PropiedadesTrait {
            public $x = 1;
        }

Las propiedades no pueden contener funciones de ningun tipo

Esto cusara un error fatal.


        trait Globals {
            public $plugin_path = "Hola";
            public $admin_page = dirname(__FILE__);
            public $plugin_url;
            public $plugin_name;
        }
