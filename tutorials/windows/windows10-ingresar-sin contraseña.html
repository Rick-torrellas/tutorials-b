Como ingresar a una seccion protegida con contraseña Windows 10

  • Estando en la pantalla de inicio, vamos a reiniciar el ordenador manteniendo precionado la tecla shift

  • Acontinuacion nos aparecera un menu de color azul:

    • Solucionar Problemas
    • Opciones Avanzadas
    • Ver mas opciones de recuperacion
    • Recuperacion de imagen de sistema
  • Volver a crear una imagen de equipo - CANCELAR
  • Selecionar una imagen del sistema
  • Opciones avanzadas
  • Instalar un controlador
  • Agregar controladores - aceptar
  • Vamos a C:\windows\system32\ (o la unidad donde tengas el sistema operativo :D)
  • Buscamos el archivo "Utilman" y le cambiamos el nombre, cualsea(una aplicacion que ayuda a personas discapacitadas, que se activa en la pantalla de inicio)
  • Luego buscamos cmd y le colocamos Utilman
  • Salimos de la instalacion de la imagion cancelando todo
  • En el menu azul le damos a continuar
  • Ahora gracias a los cambios que hicims, en la pantalla de inicio, en el icono de accesibilidad, el programa por defecto(utilman) ahora es remplazado por una consola de cmd
  • net user (para identificar el usuario admin)
  • net user usuarioAdministrador * (para cambiar el password del admin)
  • Podemos poner una nueva contraseña o darle enter para dejarlo sin contraseña
  • Ahora le damos click al usuario que se quiere acceder y listo. :D
  • NOTA Recordar volver a renombrar los archivos, con su nombre normal, sobre todo cmd
