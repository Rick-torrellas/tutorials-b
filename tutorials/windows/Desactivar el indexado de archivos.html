Desactivar el indexado de archivos

Seleccionar en que carpeta se va a desactivar el indexado

  • En el menu de inicio seleccionar Indexing Options.
  • Aqui vamos a seleccionar que carpetas queremos indexar, eso depende de ti.

Desabiitar el indexado en Unidades espesificas

  • Nos vamos a Mi Pc, en la lista de discos/unidades se daremos a propiedades
  • Desmarcaremos la casilla que dice Allow files on this drive to have contents indexed in addition to file properties y aplicar
  • Nos dara dos opciones usar Apply changes to Drive X:\, subfolders and files. y le damos a Ok

  • Buscamos la opcion Windows Search
