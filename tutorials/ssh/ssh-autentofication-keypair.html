Crear la carpeta para guardar las llaves y darle los permisos.

mkdir ~/.ssh && chmod 700 ~/.ssh

upload key al servidor ssh

windows to ubuntu

scp $env:USERPROFILE/.ssh/id_rsa.pub root@ip-adres:~/.ssh/authorized_keys

ubuntu a ubuntu

ssh-copy-id root@ip-adres

mac

scp ~/.ssh/id_rsa.pub root@ip-adres:~/.ssh/authorized_keys

lockdown logins

sudo nano /etc/ssh/sshd_config

Configuraciones


        Port el que quieras
        AddressFamily intet
        PermitRootLogin no
        PasswordAuthentication no
