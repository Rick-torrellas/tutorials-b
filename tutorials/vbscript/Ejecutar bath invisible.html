Ejecutar un bath invisiblemente hablando

Con este codigo se logra ejecutar un batch de forma que no se vea la consola de ejecucion, cabe destacar que el wscript.shell sirve para ejecutar todo tipo de programa y que la caracteristica de "invisibilidad" se lo da el atributo vbhide

set objshell = createobject("wscript.shell")
objshell.run "escrip.cmd " & id & "" ,vbhide

Tambien se podria usar esta syntaxis, es lo mismo

set WshShell = WScript.CreateObject("WScript.Shell")
Return = WshShell.Run("cmd.exe /C pm2 start G:\taller\x\x-apirest\src --name x",0,true)
