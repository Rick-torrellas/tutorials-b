Como usar env

Basico

  • Primero tienes que crear un archivo .env en la raiz de tu proyecto, creo que puede ser cualquier cosa .env pero, puedes solo colocar .env
  • Lo siguiente es en cada lugar donde quieras que exista esa variable, tienes que agregar
    require('dotenv').config();

  • La manera de editar este archivo .env sera:

    DB_HOST=localhost
    DB_USER=root
    DB_PASS=s1mpl3
  • Para utilzar la variable:
    process.env.variable
