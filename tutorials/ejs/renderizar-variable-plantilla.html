Renderizar el contenido de una variable en la plantilla

Primero que nada ejs no se parece tanto a angular, se parece mucho mas a php, por ejemplo ejs no tiene ninguna coneccion con js, las variables creadas en js no pueden ser usadas en ejs.

Para definir variables internas dentro de la platilla, tienen que ser definidos dentro etiquetas ejs, '<%' esta en espesifico se usa para colocar codigo que no va a ser rendericado como un output, aqui se hace la logica que si variables if, for, etc.

Para renderizar el conteido se usara la etiqueta <%=.

<% let nalga = 'uno'%>
<p> <%= nalga %>
<script>
let nalga = 'dos';
</script>

El resultado sera 'uno'

Se pueden usar include como php, para agregar codigo ejs externo.

Otras maneras

Con express se puede renderizar una platilla y mandar objetos junto a ella, para que la platilla pueda usar ese objeto.

La otra es declarandolo directamente en express, en las variables locales. Esto puede hacerse estaticamente, osea delcarar un valor unico a una variable local o se puede usar dinamicamente, por ejemplo se puede igualar el valor de una busqueda a una db al valor de la variable, etc.
