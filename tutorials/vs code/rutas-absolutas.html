Al colocar un archivo en la rais de tu proyecto, puedes definir la carpetas de tus comoponentes, para marcarlos como rutas absolutas.

jsconfig.json


        {
            "compilerOptions": {
              "baseUrl": ".",
              "paths": {
                "@/components/*": ["components/*"],
                
              }
            }
          }

en una aplicacion podriamos importar los componentes usando.


        import { Carta__Product } from "@/components/Carta";
