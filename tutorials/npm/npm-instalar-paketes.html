Instalar paquetes en node usando git

Para conseguir esto, simplemente tenemos que agregar el npmbre del paquetes, en alguna dependencia colocandole, el nombre que quieras, y la url del repositorio en github.

NOTE: no se que configuracion deberia tener el repositorio para que funcione, por que lo intente, con uno propio y no funciono.

xample:

"dependencies": {
"express": "^4.17.1",
"regina": "https://github.com/braintree/jsdoc-template.git"
}
