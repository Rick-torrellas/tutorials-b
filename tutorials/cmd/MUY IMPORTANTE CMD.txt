exit /b , se utiliza para salir de una subroutine o un bloque de codigo, estos bloques pueden ser for o codigo encerrado en una etiqueta :etiqueta, esto te permite crear una manera de salir de ese entorno para que el script continue ejecutandose, justo despues de donde se llamo, mira el siguiente ejemplo

@echo off
echo hola
call :hola
echo estas
pause
:hola
echo como
exit /b
echo Mierda
pause

La etiqueta :hola
Lo que se va a imprimir por pantalla sera hola como estas, y se va a oviar el "Mierda", ya que se esta saliendo es del:hola y no del script, este comportamiento es similar a tener una funcion dentro del script, brutal
