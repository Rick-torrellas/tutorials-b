Cambiar la forma de autentificacion desde mysql

Vamos a cambiar el metodo de autentificacion de mysql, a uno de contrasena y usuario. Por que es la manera mas optima.

  • Verificar la forma de autentificacion

    select user,plugin from mysql.user where user in ('root')

    Para ver todos los usuarios

    select user,plugin from mysql.user

  • cambiar el metodo de autentificacion

    alter user 'root'@'localhost' indentified with mysql_native_password by 'contrasena'
