Interfaces

El concepto de interfaces viene de typescript y se resumen en defnir un modelo de que esperar del dato, si va a hacr una string o un string de array, etc. hasta las funciones, definiendo los valores de los parametro o algo asi. Este concepto es muy parecido al de las clases y tienen sus peque√±as diferencias.

Para crear una interface, puedes hacerlo con el comando:

ng g interface <interface-name>
interface Cliente {
nombre: String;
cif: String;
direccion: String;
creado: Date;
}

Para usar la interface es tan sensillo como hacer esto:
