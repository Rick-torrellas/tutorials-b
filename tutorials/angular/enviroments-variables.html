Las enviroments variables son variables que estaran activadas segun sea su entorno de ejecucion, para produccion o desarrollo.

Para configurar estas variable hace falta ir a la raiz del proyecto eh ir a la carpeta: src/environments. Ahi encontraras dos archivos environment.prod - environment.

Esto creara un objeto con las propiedades que quieras usar en la aplicacion. Tambien se puede crear otros objetos fuera del objeto que te dan por defecto, para luego exportarlo normalmente.

export const environment = {
production: true
};
export const Nalga = {nombre: "Pedro Ramirez"}

Importante: El nuevo objeto que cree, tiene que ser creado en los dos eviroments o si no sera leido. Nota: Es importante tener los mismos valores en ambos archivos, puede afectar ciertos procesos que necesiten esos valores.

Para importarlos a un componente o lo que sea:

import {environment,Nalga} from '../../../environments/environment';
