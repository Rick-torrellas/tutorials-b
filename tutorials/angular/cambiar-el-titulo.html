Cambiar el titulo en una aplicacion en Angular

app.module.ts

Lo primero que hay que hacer es importar el componente de titulo al module.ts el cual se encuentra en src/app/app.module.ts

import { BrowserModule, Title } from '@angular/platform-browser';
@NgModule({
providers: [
Title
],
})

tu-componente.component.ts

Vamos a importarlo y usar los metodos disponibles, no se tiene que usar ajuro en ngOnInit()

import { Title } from '@angular/platform-browser';
public constructor(private titleService: Title ) { }
ngOnInit() {
this.titleService.setTitle( newTitle );
}
