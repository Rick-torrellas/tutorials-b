Angular Universal

Creando componente

Si estas usando angular universal y tratamos de crear un componente nos lanzara el siguiente error:

More than one module matches. Use skip-import option to skip importing the component into the closest module.

Para corregir esto:

ng g c new-component --module app
o si estas en otro directorio
ng g c component-name --module ../app
(sin probar ng generate component componentName --module=app.module)

Instalacion

ng add @nguniversal/express-engine
