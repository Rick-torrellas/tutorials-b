Colocar un componente dento de otro

En este ejemplo se colara un componente dentro de otro, creando una relacion padre hijo, el componente padre a su vez podra pasarle informacion

Para lograr esto siemplemente hay que llamar el elemento hijo dentro del template del elemento padre

<h2>Elemento padre</h2>
<app-hijo></app-hijo>
<p>Contenido del elemento padre</p>
