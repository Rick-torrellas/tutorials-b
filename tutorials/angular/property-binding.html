Primero hay que aclarar 1 concepto, hay que saber que existen attributos html y propiedades del DOM.

Los valores de los atributos html no se pueden cambiar durante la ejecucion solo cuando se escribe el html, en cambio las propiedades que referencian a ese atributo en el dom si se pueden cambiar.

Cuando hablamos de binding propiedades, hablamos de usar la interpocalacion para asignale un valor a ese atributo. En este caso se cambia es la propiedad del dom no el atributo html.

Para lograr el binding colocamos la siguiente syntax:

<p [id]="myId">Hola</p>

Tambien hay que recordar que esta exprecion tambien se puede usar como una condicional, por ejemplo se podria decir myId.length == 0, en caso de un array o un objeto y aplicarlo en caso de tener un length.
