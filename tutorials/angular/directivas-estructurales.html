*ngIf

Esta directiva esrtuctural funciona muy parecido al if de javascript, la diferencia aqui es que este condicionara si se mostrara o no el contenido, el if se va puede ayudar de varios componentes, como el ng-template, ng-container, ngTemplateOutlet, los cuales ayudan a crear y encapsular contenido dependendo de la condicion, tambien puede guardar el resultado de la condicional en una variable con as y se puede tambien usar una sentencia else que servira para renderizar un ng-template y un then en caso de querer espesificar donde ir en caso de la condicion sea correcta.

La diferente entre usar este y el attributo bidding [hidden]="condicion", es que con ngIf el elemento no es renrerizado, mientras que con el atributo lo que logramos que no se muestre pero sigue existiendo.

Muy importante es importante recordar que se puede usar el if para comparar string con ''. pepito: string='vaginas';

<p class="blockquote text-center" *ngIf="pepito == 'vagina'"></p>

Ejemplo Basico

<div *ngIf="condicion.length > 0">
<p>Contenido a mostrar si la condicion es true</p>
</div>

Con un else

<div *ngIf="condicion; else nalga">
<p>Contenido a mostrar si la condicion es true</p>
</div>
<ng-template #nalga>
<p>Contenido se mostrara en caso de ser false</p>
</ng-template>

Un ng-template con un if

<ng-template [ngIf]="condition"><div>Content to render when condition is true.</div></ng-template>

Con un else y un then

<div *ngIf="condition; then thenBlock else elseBlock"></div>
<ng-template #thenBlock>Content to render when condition is true.</ng-template>
<ng-template #elseBlock>Content to render when condition is false.</ng-template>

Guardando el resultado de la condicion en una variable

<div *ngIf="condition as value; else elseBlock">{{value}}</div>
<ng-template #elseBlock>Content to render when value is null.</ng-template>

Usando el ng-container

Este nos permite encapsular un condicion y dentro colocar otra directiva de estructura, y tambien nos permite colocar un ngTemplateOutlet, para referenciar un ng-template y asi re utilizarlo.

Usando ng-container

<ng-container *ngIf="condicion; else nalga">
<div *ngFor="let culo of culos">
{{culo.shape}}
</div>
</ng-container>
<ng-template #nalga>
<p>Contenido se mostrara en caso de ser false</p>
</ng-template>

Usando ngTemplateOutlet

Esto renrerizara el ng-template

<ng-container *ngTemplateOutlet="nalga"></ng-container>
<ng-template #nalga>
<p>Contenido se mostrara en caso de ser false


</ng-template>

*ngFor

Esta sirve para interar en un array on un objeto y asi crear elementos en la plantilla.

<div *ngFor="let nalga of nalgas">
<p>{{nalga.name}}</p>
</div>

*ngSwitch

<div [ngSwitch]="condicion">
<p [*ngSwitchCase="1"]></p>
<p [*ngSwitchCase="2"]></p>
<p [*ngSwitchCase="3"]></p>
<p [*ngSwitchCase="4"]></p>
<p [*ngSwitchCase="5"]></p>
</div>
