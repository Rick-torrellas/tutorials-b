Referenciar un elemento html

Estas variables nos permiten referenciar al elemento html y todas sus propiedades del dom:

<p #myVariable></p>

Hay que saber que esta propiedad se carga cuando se carga la vista, asi que en vez de onInit() se puede colocar ngAferViewInit(), acordarse se exportarlo y la classe del componente en vez de implementar OnInit va a implementar AfterViewInit, pero eso defependiendo que vallas a usar la funcion al iniciar.

Para realizar la aplicacion hay que importar ElementRef - ViewChild del @angular/core

Para extraer el valor del elemento se usa el decorador

@ViewChild('myVariable') myVariable: ElementRef;

ngAfterViewInit() {
this.myVariable.innerHTML = "Vagina";
}
