Crear un componente

Para crear nuevos componentes se tienen que crear dentro de app

nombre-componente.component.ts

siempre se empieza cargando el core de angular

import { Component } from '@angular/core';
@Component({
selector: 'mi-componente',
template: `
<p>codigo html</p>
`
templateUrl: 'archivo-html.component.html'
})
export class Micomponente{
constructor() {
console.log("Componente cargado");
}
}

para cargar este nuevo componente hay que primero, importarlo en app.modulo.ts

import { Micomponente } from './componentes/mi-componente/mi-componente.component'

@NgModule({
declarations: [
AppComponent,
Micomponente
]
})

Luego para utilizar tu componente en app.component.html

se usa dentro del codigo html con el nombre que le pusiste

<mi-componente></mi-componente>

Para usar las propiedades de una clasepor ejemplo, dentro del html, se puede usar con binding interpolation angular, con lo que se puede aregar propiedades y metodos. asi.

{{ propiedad }}
{{ metodo }}

para hacer automaticamente todo lo que acabamos de ver, podemos simplemente usar el siguiente comando.

ng g component vaginas /mi-componente
ng g c vaginas /mi-componente
