Actualizar la Vercion de Angular de tu Proyecto

Primero se debe instalar la ultima vercion de angular en el proyecto con:

npm i @angular/cli@latest

Luego tocara actualizar las dependencias del proyecto con:

ng update @angular/cli

Luego se actualizan las librerias con:

ng update

En este cao el mismo angular te indicara que necesita actualizar, y te dara los comandos para hacerlo uno por uno, pero si se quiere actualizar todo de un vergazo:

ng update --all
