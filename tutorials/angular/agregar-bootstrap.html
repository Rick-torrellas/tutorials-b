Para instalar bootstrap en tu proyecto de angular, se puede hacer de las maneras tradicionles, ya sea con la CDN, o descargando los archivos manualmente, pero la manera mas practica es instalarlo con npm.

npm

Instalar bootstrap

npm i bootstrap

Instalar jquery

npm i jquery

Instalar popper.js

npm i @popperjs/core

Ahora bien, para implementar bootstrap en tu proyecto, se puede hacer de la manera convencional, en tu index.html, y la otra desde angular.json.

Aplicar bootsrap manera convencional

<head>
<link rel="stylesheet" href="node_modules\bootstrap\dist\css\bootstrap.min.css">
</head>
<body>
<script src="node_modules\jquery\dist\jquery.min.js"></script>
<script src="node_modules\@popperjs\core\dist\umd\popper.min.js"></script>
<script src="node_modules\bootstrap\dist\js\bootstrap.min.js"></script>
</body>

Aplicar bootsrap en angular.json

"styles": [
"src/styles.css",
"./node_modules/bootstrap/dist/css/bootstrap.min.css"
],
"scripts": [
"./node_modules/jquery/dist/jquery.min.js",
"./node_modules/@popperjs/core/dist/umd/popper.min.js",
"./node_modules/bootstrap/dist/js/bootstrap.min.js"
]
