Rutas

Se pueden definir las rutas en angular de 3s maneras:

1 - Directamente en el app-module.ts:

2 - Se puede crear un archivo especial: puedes usar ng g module nombre-module

3 - Dejar que angular lo cree por ti:

En las 3s se va respetar el mismo principio.

import { NgModule } from '@angular/core';
import { Routes, RouterModule } from '@angular/router';


const routes: Routes = [];

@NgModule({
imports: [RouterModule.forRoot(routes)],
exports: [RouterModule]
})
export class AppRoutingModule { }

Y por ultimo en la parte del imports del app-module.ts:

Hay que agregar el archivo con las rutas o directo si se hace directo

RouterModule.forRoot(appRoutes)
